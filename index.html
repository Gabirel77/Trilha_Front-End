Parágrafos e textos

Este é um parágrafo de texto. Ele é usado para organizar o conteúdo em blocos de texto legíveis. Dentro de um parágrafo, você pode usar várias tags para formatar o texto, como negrito, itálico, e sublinhado ou riscado. Além disso, você pode criar listas, links e outros elementos para enriquecer o conteúdo do seu site.

Também é possível marcar trechos quase como um marca-texto ou usar a tag <code> para destacar código ou comandos. As tags de formatação ajudam a tornar o conteúdo mais atraente e fácil de ler para os visitantes do seu site. Textos pequenos podem ser colocados aqui ou até escrever equações químicas ou matemáticas usando a tag subscrito e sobrescrito.

Listas

Existem dois tipos principais de listas em HTML:

Lista ordenada

As listas ordenadas são usadas quando a ordem dos itens é importante. Elas são criadas usando a tag <ol> e cada item da lista é definido usando a tag <li>. Por exemplo: 

          
            <ol>
                <li>Primeiro item</li>
                <li>Segundo item</li>
                <li>Terceiro item</li>
            </ol>

Lista não ordenada

As listas não ordenadas são usadas quando a ordem dos itens não é importante. Elas são criadas usando a tag <ul> e cada item da lista é definido usando a tag <li>. Por exemplo: 

            
                <ul>
                    <li>Item A</li>
                    <li>Item B</li>
                    <li>Item C</li>
                </ul>

Exemplo lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplo lista NÃO ordenada:

  • Item 1
  • Item 2
  • Item 3
